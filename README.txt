Requirements:
Python >= 3.4
PyTorch >= 1.0.1
matplotlib, numpy, scipy, sklearn

Note: Since there are some animated plots in the demo, please don't use 'inline plotting'. Instead, configure your favourite Python IDE to open a new window for displaying plots.

1) MNIST: Run "MNIST_gen.py" file to run experiements related to MNIST. By default this loads the pre-trained model.
	  Incase, one wishes to train the model from scratch, run "RKM_MNIST.py" file. 
          Other settings could be adjusted in the 'utils.py' file.
